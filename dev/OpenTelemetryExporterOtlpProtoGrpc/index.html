OpenTelemetryExporterOtlpProtoGrpc

This package uses OpenTelemetryProto to communicate with OTEL Collector through gRPC.

OpenTelemetryExporterOtlpProtoGrpc.OtlpProtoGrpcTraceExporter — Method
OtlpProtoGrpcTraceExporter(;kw...)

Keyword arguments

  • scheme=http
  • host="localhost"
  • port=4317
  • is_blocking=true, by default the BlockingClient is used.
  • Rest keyword arguments will be forward to the gRPC client.

scheme, host and port specifies the OTEL Collector to connect with.
